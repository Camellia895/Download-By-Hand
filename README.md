觉得手动下载x的图像很麻烦，网络上的脚本要不是就是需要付费，要不就是太过手工业。不悦。于是用gemini写了个脚本，还算好用（有问题尽管提便是，我大致知道代码的运行逻辑所以不至于不懂）。

在推文中添加“下载”按钮，下载PNG原图。在用户的'媒体'页面，增加“批量下载”按钮，可滚动收集页面所有图片后一键下载。
